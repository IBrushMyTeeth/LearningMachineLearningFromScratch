# Logistic Regression

This folder contains several projects exploring logistic regression with PyTorch, covering both binary and multi-class classification.

## Overview
These experiments focus on understanding logistic regression as a probabilistic model, how PyTorch handles automatic differentiation, and how loss functions like binary cross entropy and cross entropy connect to maximum likelihood. The projects build from binary classification to multi-class settings, with an emphasis on clarity, stability, and proper evaluation.

- Evaluated models using confusion matrices, ROC curves, and class-wise performance.
